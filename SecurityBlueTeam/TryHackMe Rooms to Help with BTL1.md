# TryHackMe Rooms to Help with BTL1
This guide provides a list of TryHackMe rooms that can help you study for and pass the BTL1 exam.

## Phishing
- [Phishing Module](https://tryhackme.com/module/phishing): This module contains 5 rooms (2 free, 3 paid/VIP) that walk you through analysis of phishing emails, tools you can use to analyze phishing emails, and phishing prevention techniques.
    1. [Phishing Analysis Fundamentals](https://tryhackme.com/room/phishingemails1tryoe)
    2. [Phishing Emails in Action](https://tryhackme.com/room/phishingemails2rytmuv)
    3. [Phishing Analysis Tools](https://tryhackme.com/room/phishingemails3tryoe)
    4. [Phishing Prevention](https://tryhackme.com/room/phishingemails4gkxh)
    5. [The Greenholt Phish](https://tryhackme.com/room/phishingemails5fgjlzxc)
